## how to use

- make "vendor" directory
- git clone to vendor dir [ncmb-ruby-client](https://github.com/moongift/ncmb-ruby-client)
- create settng.yml
 - set ncmb api key
 - set mqtt server config

```
application_key: xxxxxxx
client_key: xxxxxxx
username: xxxxxx
password: xxxxxx
hostname: xxxxx.xxxxxx.xxxx.xxx
portnumber: '00000'
topic: xxxxxxx
```
